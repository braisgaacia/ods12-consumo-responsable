# ODS 12 - Producción y Consumo Responsables

Sitio web de ejemplo sobre el **Objetivo de Desarrollo Sostenible 12** de la Agenda 2030.

## Estructura
- `index.html`: introducción y metas del ODS 12.
- `secundaria.html`: datos destacables y noticias del ODS 12.
- `style.css`: estilos compartidos para menú y contenedores.

## Requisitos implementados
- Dos páginas (`index.html` y `secundaria.html`).
- Menú con 5 enlaces en cada página (4 internos + 1 a la otra página).
- Cuatro contenedores en cada página con estilo consistente:
  - color de fondo,
  - borde,
  - padding,
  - márgenes.
- Contenido específico solicitado:
  - imágenes,
  - texto con encabezados y párrafos,
  - dos imágenes intermedias en el bloque de texto,
  - listas con enlaces externos,
  - infografía (index),
  - vídeo (secundaria).
